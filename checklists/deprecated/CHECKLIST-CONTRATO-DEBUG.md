# CHECKLIST — CONTRATO DE DEBUG

Este checklist valida que a atividade executada
foi **exclusivamente de investigação**, sem correção.

---

## Escopo

- [ ] Nenhum arquivo de código foi alterado
- [ ] Nenhum seed foi criado ou ajustado
- [ ] Nenhuma permissão foi modificada
- [ ] Nenhum teste foi criado ou alterado
- [ ] Nenhum commit de correção foi realizado

---

## Análise

- [ ] O problema foi descrito objetivamente
- [ ] Evidências foram coletadas (logs, código, config)
- [ ] Hipóteses foram claramente separadas de fatos
- [ ] Causa raiz foi identificada ou marcada como indeterminada

---

## Saída

- [ ] Plano de correção sugerido foi apresentado
- [ ] Contrato correto para correção foi indicado
- [ ] Nenhuma ação foi executada fora do Debug

---

## Declaração Final

- [ ] Esta atividade foi exclusivamente de DEBUG
- [ ] Nenhuma correção foi aplicada
- [ ] A próxima ação exige troca de contrato
